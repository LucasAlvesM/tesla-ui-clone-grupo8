### Tools used

- React
- TSX e TS
- HTML, CSS e JS

### To run the dependencies it is necessary to run the following commands:
 - ``` npx create-react-app --template=typescript clone-tesla ```
 - ``` yarn add framer-motion styled-components ```
 - ``` yarn add -D@types/styled-components ```
